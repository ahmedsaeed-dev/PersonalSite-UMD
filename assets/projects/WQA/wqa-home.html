The Website Quality Assessment is a tiny tool used to measure the quality of an existing or proposed website as compared to the general website quality model described in Pressman and Maxim's Software Engineering: A Practitioner's Approach.

The Website Quality Assessment uses a Q&A format with different rankings to help the user determine the quality of a given website on a scale from 1 to 5, with 5 defined as Strongly Agree, 1 defined as Strongly Disagree, and a choice for Non-Applicable.

If you need help, click on the question mark in the bottom right hand corner! It's found on all pages!

Click the Begin Assessment button to begin the assessment process.
